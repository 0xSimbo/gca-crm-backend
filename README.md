# Elysia with Bun runtime

## Setup

1. Change the package name in `package.json` to your project name.
   - This will be the name used for the SDK

## Tools

1. [Elysia](#https://elysiajs.com/)
   - An end to end typesafe API client
2. [Planetscale](#https://planetscale.com/)
   - A serverless database for MySQL
3. [Drizzle](#https://orm.drizzle.team/)
   - A typesafe edge compatible ORM

## Development

To start the development server run:

```bash
bun run dev
```
